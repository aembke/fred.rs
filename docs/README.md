Extra Documentation
===================

This library exposes some interfaces that are difficult to describe in rust docs. These documents outline some of these implementation details and provide extra information on when and how to use some of the more complicated features.

* [Cluster TLS](./cluster-tls.md) - Describes how to configure more advanced TLS settings in the context of a clustered deployment.
* [Features](./features.md) - Describes some of the more complicated features in the client.
