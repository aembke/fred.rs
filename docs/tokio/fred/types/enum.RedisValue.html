Enum fred::types::RedisValue

source · 
pub enum RedisValue {
    Boolean(bool),
    Integer(i64),
    Double(f64),
    String(Str),
    Bytes(Bytes),
    Null,
    Queued,
    Map(RedisMap),
    Array(Vec<RedisValue>),
}
Expand description

A value used in a Redis command.

Variants§

§

Boolean(bool)

A boolean value.

§

Integer(i64)

An integer value.

§

Double(f64)

A double floating point number.

§

String(Str)

A string value.

§

Bytes(Bytes)

A byte array value.

§

Null

A nil value.

§

Queued

A special value used to indicate a MULTI block command was received by the server.

§

Map(RedisMap)

A map of key/value pairs, primarily used in RESP3 mode.

§

Array(Vec<RedisValue>)

An ordered list of values.

In RESP2 mode the server usually sends map structures as an array of key/value pairs.
